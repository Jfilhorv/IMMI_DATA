## Scripts used for updates (all exist in `scripts/`)

| # | Script | Purpose |
|---|--------|---------|
| 1 | `fetch_first_dataset.py` | Download XLSX from data.gov.au; save to `data/raw/`; export each sheet to `data/by_sheet/*.csv`; write `INDEX.csv`, `first_dataset_summary.json` |
| 2 | `melt_all_sheets.py` | Read `data/raw/*.xlsx`; write `data/by_sheet/` again + `data/melted/*.csv` (long format) |
| 3 | `build_dashboard_data.py` | Read `data/melted/`; write `data/dashboard/`: `indicators.csv`, per-table CSVs, `category_tables.json`, `indicator_metadata.csv` |
| 4 | `extract_table_footnotes.py` | Read `data/by_sheet/`; write `data/dashboard/table_footnotes.json` and copy to `dashboard/data/table_footnotes.json` |
| 5 | `list_kpi_candidates.py` | Read `data/dashboard/indicators.csv`; write `data/dashboard/kpi_candidates.json` + `kpi_candidates_list.md`; copy JSON to `dashboard/data/kpi_candidates.json` |
| 6 | `sync_dashboard_data.py` | Copy `data/dashboard/` → `dashboard/data/` (indicators.csv, *.csv, category_tables.json, indicator_metadata.csv) so the live site has all files |
| 7 | `build_drawio_tree.py` | Optional. Read `dashboard/data/tables.json` + `indicators.csv`; write `docs/indicator_tree.drawio` and `indicator_tree.drawio.html` |

**Note:** `tables.json` in `dashboard/data/` is not generated by these scripts. If the new XLSX has different sheets/table list, update `dashboard/data/tables.json` manually (section, id, title, shortTitle per table). `build_index_csv.py` and `export_sheets_to_md.py` are for docs/export only and are not required for the dashboard update.

## How to update (new financial year)

1. **Get the new resource URL**  
   Open [data.gov.au – Australian Migration Statistics](https://data.gov.au/data/dataset/australian-migration-statistics) and find the new year’s XLSX (e.g. `migration_trends_statistical_package_2025_26.xlsx`). Copy its download URL.

2. **Point scripts at the new file**
   - In `scripts/fetch_first_dataset.py`: set `DATASET_1_URL` to the new URL; set `xlsx_name` (e.g. `migration_trends_statistical_package_2025_26.xlsx`).
   - In `scripts/melt_all_sheets.py`: set `RAW_XLSX` to the path of that XLSX in `data/raw/` (same filename as above).

3. **Run the full pipeline** (from project root):

   ```bash
   pip install -r requirements.txt
   python scripts/fetch_first_dataset.py
   python scripts/melt_all_sheets.py
   python scripts/build_dashboard_data.py
   python scripts/extract_table_footnotes.py
   python scripts/list_kpi_candidates.py
   python scripts/sync_dashboard_data.py
   python scripts/build_drawio_tree.py
   ```

   Or run the single entry point (if you use it):

   ```bash
   python scripts/run_update.py
   ```

4. **If the table list changed**  
   Edit `dashboard/data/tables.json` to add/update table entries (id, section, title, shortTitle).

5. **Commit and push**  
   Commit updated `data/` and `dashboard/data/` (and `docs/` if you ran `build_drawio_tree.py`), then push. GitHub Pages will serve the updated dashboard.
