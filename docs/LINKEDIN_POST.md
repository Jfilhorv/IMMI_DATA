Australian migration data is public, but it's not always easy to explore or compare across years and categories.

To make this easier, I built an open dashboard using official Australian Government data (Home Affairs / data.gov.au). The project collects, normalises, and visualises migration statistics, making them easier to explore interactively.

The dashboard currently allows you to explore 577 combinations of sections, tables, and indicators from the official dataset.

You can browse the full index here:
https://jfilhorv.github.io/IMMI_DATA/dashboard/tree.html

**Data coverage.** All metrics follow the aggregation levels published in the official dataset.
