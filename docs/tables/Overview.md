|  |  |
|  | Overview |
|   | Australian Migration Statistics is produced as an accompaniment the publication Australia's Migration Trends.   |
|  | Detailed tables are provided summarising migration activity for the financial year including 10-year comparative data.  |
|  | Coverage includes permanent and temporary migration, citizenship, Net Overseas Migration and labour market tables. |
|  |  |
|  | Report Period and Frequency |
|  | 2024–25 financial year, published annually. |
|  |  |
|  | Caveats |
|  | This Australian Migration Statistics dataset provided by the Department of Home Affairs is for statistical and research purposes. |
|  | Any publications or reports using this data must cite the Department as the source. |
